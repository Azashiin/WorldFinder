# Notices

World Finder is a client-side seed map mod for Minecraft.

## Project License

World Finder is licensed under `LGPL-3.0-or-later`.

## Third-Party Components

### xpple/SeedMapper

This project adapts seed-map execution/cache design ideas and generated Java binding patterns from `xpple/SeedMapper`.

- License: `LGPL-3.0-or-later`
- Source: https://github.com/xpple/SeedMapper

### xpple/cubiomes

This project uses Java bindings based on `xpple/cubiomes`.

- License: `MIT`
- Source: https://github.com/xpple/cubiomes

### Cubitect cubiomes

The seed-based biome and structure calculations are backed by cubiomes concepts and native logic.

- License: `MIT`
- Source: https://github.com/Cubitect/cubiomes

## Optional Mod Integrations

WorldFinder detects JourneyMap, Xaero's Minimap/World Map, and FTB Chunks at runtime through optional reflection-based integrations. These mods are not bundled with WorldFinder.
